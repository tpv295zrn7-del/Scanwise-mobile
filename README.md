# ScanWise Mobile

React Native (Expo) mobile application for ScanWise.

## Setup

```bash
npm install
```

## Run locally

```bash
npm start
npm run android
npm run ios
npm run web
```

## Quality checks

```bash
npm run lint
npm run type-check
npm test
```

Coverage is enforced globally at 90%+ for statements, branches, functions, and lines.
